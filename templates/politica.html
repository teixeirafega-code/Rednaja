Política de Privacidade

Última atualização: 2026

1. Apresentação

Esta Política de Privacidade descreve como o MEI Organizado coleta, utiliza, armazena e protege os dados pessoais fornecidos pelos usuários ao utilizar a plataforma.

Ao utilizar o serviço, o usuário concorda com os termos aqui descritos.

2. Dados Coletados

Podemos coletar os seguintes dados:

a) Dados fornecidos pelo usuário

  • Nome
  • E-mail
  • Arquivos enviados (extratos bancários, relatórios de vendas)
  • Informações financeiras contidas nesses arquivos

b) Dados coletados automaticamente

  • Endereço IP
  • Informações de navegador
  • Dados de uso da plataforma

3. Finalidade do Tratamento

Os dados são utilizados para:

  • Processamento das análises financeiras
  • Identificação de divergências
  • Geração de relatórios
  • Melhoria da experiência do usuário
  • Comunicação relacionada ao serviço

Não utilizamos dados financeiros para venda ou compartilhamento comercial.

4. Base Legal (LGPD)

O tratamento dos dados ocorre com base em:

  • Consentimento do usuário
  • Execução de contrato (prestação do serviço solicitado)
  • Cumprimento de obrigações legais, quando aplicável

5. Compartilhamento de Dados

O MEI Organizado não vende dados pessoais.

Os dados poderão ser compartilhados apenas com serviços essenciais para funcionamento da plataforma (hospedagem, armazenamento e infraestrutura), sempre com medidas adequadas de segurança.

6. Armazenamento e Segurança

Adotamos medidas técnicas e organizacionais para proteger os dados contra acesso não autorizado, perda, alteração e divulgação indevida.

Apesar disso, nenhum sistema é totalmente imune a falhas ou ataques.

7. Direitos do Usuário

Nos termos da LGPD, o usuário pode solicitar:

  • Confirmação da existência de tratamento
  • Acesso aos dados
  • Correção de dados incompletos
  • Exclusão dos dados
  • Revogação do consentimento

Solicitações podem ser feitas pelo e-mail de contato informado pela plataforma.

8. Retenção de Dados

Os dados serão armazenados apenas pelo tempo necessário para cumprir a finalidade do serviço, atender obrigações legais e resolver eventuais disputas.

9. Alterações na Política

Esta Política poderá ser atualizada a qualquer momento. Recomendamos revisão periódica deste documento.

10. Contato

Em caso de dúvidas relacionadas à privacidade e proteção de dados, entre em contato pelos canais oficiais da plataforma.
